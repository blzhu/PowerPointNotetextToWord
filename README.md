# PowerPointNotetextToWord
PPT Notes Export Tool - User Manual

This software is used to extract the "Notes" section from PowerPoint presentations (PPT) and export them to a Word document with one click. The interface is simple and mainly includes the following function buttons.

Core Function Buttons

1. Browse and export PPT files

Purpose: Process PPT files stored on the computer's hard drive (no need to open the PPT in advance).

When to use:

When the PPT file you want to export is still sitting quietly in a folder and not opened.

After clicking, a file selection window will pop up. Locate the .pptx file you want to process and double-click to automatically start the export.

2. Export selected presentation notes to Word

Purpose: Operate on the PPT file you are currently editing or have opened.

When to use:

When you have one or more PPT files open and viewing, and you suddenly want to export the notes from the current PPT.

Steps:

In the "Currently Open Presentations" list above, left-click to select the PPT you want to export.

Click this button and the software will immediately operate on the selected PPT.

3. Refresh the list

Purpose: Update the "Currently Open Presentations" list.

When to use:

If you open the application first and then open a PPT file, the file might not appear in the software's list immediately.

Click "Refresh List" and the newly opened PPT will appear in the list.

4. Clear List

Purpose: Clear the displayed information on the software interface.

When to use:

Use this when you have processed too many files and the logs or list information on the interface are too cluttered and you want to clean up the view (this will not close any PPT you actually have open; it only clears the software interface display).

5. Exit

Function: Closes this software.

When to use:

When all work is finished, click Exit.

Quick reference for common usage scenarios

Click [Browse and Export PPT File]

Select it in the list, then click [Export Selected Presentation Notes to Word]

PPT 备注导出工具 - 使用说明书

本软件用于将 PowerPoint 演示文稿（PPT）中的“备注”部分，一键提取并导出为 Word 文档。界面简洁，主要包含以下功能按钮。

核心功能按钮

1. 浏览并导出 PPT 文件

作用：处理电脑硬盘里保存的 PPT 文件（不需要预先打开 PPT）。

什么时候用：

当你想要导出的 PPT 文件还静静地躺在文件夹里没打开时。

点击后，会弹出一个文件选择窗口，找到你要处理的 .pptx 文件，双击即可自动开始导出。

2. 导出选中演示文稿备注到 Word

作用：处理你正在编辑或已经打开的 PPT 文件。

什么时候用：

当你已经打开了一个或多个 PPT 正在看，突然想把当前这个 PPT 的备注导出来。

操作步骤：

在上方“当前打开的演示文稿”列表里，鼠标左键点击选中你要导出的那个 PPT。

点击这个按钮，软件就会立即对你选中的这个 PPT 进行操作。

3. 刷新列表

作用：更新“当前打开的演示文稿”列表。

什么时候用：

当你先打开了本软件，然后才去打开 PPT 文件时，软件列表里可能暂时没显示出来。

点击一下“刷新列表”，刚打开的 PPT 就会出现在列表里了。

4. 清空列表

作用：清空软件界面上的显示信息。

什么时候用：

当你处理了太多文件，界面上的日志或列表信息太乱，想清理一下视野时使用（这不会关闭你实际打开的 PPT，只是清理软件界面显示）。

5. 退出

作用：关闭本软件。

什么时候用：

所有工作都做完了，点击退出。
